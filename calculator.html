Simple Calculator Project in C

BRIEF DESCRIPTION

This is a simple group project that me and my friends made for our final grade when we were in first year of college, 1st semester. My contribution to the project is a simple calculator. I included some input validation in the code so that the program doesn't accept anything non-numerical.

PHOTOS

  1. Main menu of the program.

  2. After entering number 1, the "calculator" option opens.

  3. You can now select what arithmetic operation to use.
